Director

an R package for dynamic, browser-enabled visualization of multi-level regulatory interactions

Summary

As datasets from high-throughput technology grow increasingly complex, novel approaches are needed to visually convey key findings quickly and effectively. Director is an R package designed to streamline the visualization of molecular effects in regulatory cascades. It utilizes a modified Sankey plugin of the JavaScript library D3 to provide a fast and easy, web-based solution to discovering potentially interesting downstream effects of regulatory and/or co-expressed molecules. The diagrams are robust, interactive, and packaged as highly-portable HTML files that eliminate the need for third-party software to view.

Installation

Install Director from its GitHub repository. In R, you first need to install devtools.

install.packages("devtools")`

Then using the 'install_github' function in the devtools package:

library(devtools)
install_github("kzouchka/Director")

Example use

The following test case draws a set of regulating miRNAs and their target genes in key biological pathways using default package settings.

library(Director)
data(ovca) # import data
initSankey() # same as initSankey(directory=getwd())
Level1 <- createList(ovca$Level1) # miRNA-gene data
Level2 <- createList(ovca$Level2) # gene-pathway data.
List <- append2List(Level1, Level2) # append lists.

Sankey <- makeSankey(List, averagePath = TRUE) # calculate colour values
drawSankey(Sankey,caption="Regulatory pathway",
width=1400, height=900)
writeSankey(name="FDRpathways") # write to working directory as ’FDRpathways.html’
